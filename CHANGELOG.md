# 更新日志 / CHANGELOG

## v1.0.0

> 🎉首个正式版发布！

**新增功能**

- ✨新增**主页**界面，展示课表等功能的快捷入口。
- 📚新增**教务系统账户**的统一认证登录。
- 📚新增**课表**功能，支持从教务系统获取并缓存课程表以供查看。
- 📚新增**选课**功能，支持浏览已选和可选课程，并进行抢课或退课操作。
- 📚新增**考试**功能，支持查询本学期考试的时间和考场信息。
- 📚新增**成绩**功能，支持查询所有课程的成绩。
- 📶新增**校园网管理账户**的登录。
- 📶新增校园网账户的实时**流量监视**功能。
- 📶新增校园网账户的**自助服务**功能，包括设备管理和账单查询等功能。
- 🔁新增**同步**功能，支持跨设备同步配置数据。
- 🔈新增**公告**功能。
- ⬆️新增**更新**功能，支持查询软件版本信息和下载新版程序。
- ⚙️新增**设置**功能。

**构建**

- 🔧新增**持续集成** (CI) 工作流，并支持 Android、Windows 和 Linux 的发行文件构建。
